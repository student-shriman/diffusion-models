2. Text to Images using AI - Pornpen, Imagen, Deepai, HotPot ai

  =>  Pornpen is basically a platform for generating some hentai and animes adult contents by taking some Textual inputs.
  =>  Imagen is a state-of-theart model for generating Images from language understanding or Textual inputs. It is published by Google research team and it uses T5 model for NLU.
  =>  Deepai is an API for generating Images from Text.
  =>  hotpot.ai is a platform for generating images, removing backgrounds or foregrounds, colorizing of images, restoring the old images, upscaling and enhancing images  etc. using GANs.
